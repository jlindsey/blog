JavaScript – unlike many of the languages you might use to interact with it – deals with Unix Epoch timestamps in milliseconds instead of seconds. For instance, PHP's time(), Ruby's Time.now, and Python's time.time() all return the Epoch in seconds. This can lead to some strange behaviors if you aren't expecting it. It's also fairly annoying when passing timestamps back and forth between your server-side language and JavaScript.
